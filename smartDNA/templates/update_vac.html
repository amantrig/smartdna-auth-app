smartDNA™ Asset Validator:

Asset Code validator is an extended service of smartDNA™ cloud services to help deployments audit the tracking codes associated with smartDNA™ credential in database. Once using ADMIN privileges, the tracking-code range is specified,there onwards any out of range tracking code or duplicate tracking code will be auto-reflected in alert table along with IP address to assist in further inspection.

Valid Range:

It should be any numeric value(e.g.:100000).

Criteria for Valid Asset Code:

Asset-codes are alpha-numeric fields wherein first 3/4 characters will be brand identifier and rest numeric part is product unit identifier which should be within range (e.g.:FED098331).
